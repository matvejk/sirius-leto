Приложение для проведения цифровых лабораторных работ по физике Help

Глобальные константы

Переменные

  • os: module

    Импортировано из os

  • sys: module

    Импортировано из sys

  • perf_counter: Callable

    Импортировано из time.perf_counter

  • real_time: Callable

    Импортировано из time.time

  • g: float

    Импортировано из scipy.constants.g

  • pi: float

    Импортировано из math.pi

  • PROJECT_ROOT: str

    Корневой каталог проекта.

  • DATASTORE_ROOT: str

    Корневой каталог хранилища временных данных.

  • ASSETS_ROOT: str

    Корневой каталог хранилища внешних ресурсов.

Функции

  • sleep

def sleep(duration)

Аналог time.sleep с повышенной точностью (до 100 нс).

Аргументы

  • duration: float

Требуемая продолжительность "сна".
